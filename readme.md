# Catch Catch
Catch Catch는 집에 혼자있는 시간이 많은 반려동물을 위해 개발한 제품으로 프로젝터와 영상 처리를 이용해 반려동물이 혼자서도 즐길 수 있는 상호작용형 디지털 콘텐츠이다.
